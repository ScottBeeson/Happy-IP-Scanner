# Happy IP Scanner

Happy IP Scanner is a simple, modern desktop application for Windows built with Electron. It allows users to scan their network for active devices and includes a "Known Devices" feature to easily filter out recognized hardware and spot intruders. Fast, clean, and intuitive.

## Features

- **Fast Network Scanning**: Quickly identify active devices on your local network.
- **Known Devices**: Mark trusted devices to easily spot new or unknown hardware.
- **Modern UI**: Clean, responsive interface designed for Windows.
- **Export**: (Planned) Export results for analysis.

## Getting Started

### Prerequisites

- Node.js
- npm

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/happy-ip-scanner.git
   ```
2. Navigate to the project directory:
   ```bash
   cd happy-ip-scanner
   ```
3. Install dependencies:
   ```bash
   npm install
   ```

### Running the App

```bash
npm start
```
